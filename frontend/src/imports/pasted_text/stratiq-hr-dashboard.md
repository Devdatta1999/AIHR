🎯 Figma Prompt: Agentic HR SaaS Platform

Prompt:

Design a modern, enterprise-grade AI-native HR SaaS dashboard called “StratIQ HR”.

The design should feel premium, minimal, and data-driven, similar to tools like Workday, Rippling, or Salesforce, but enhanced with AI agent workflows.

⸻

🧩 Layout Requirements
	•	Create a responsive web dashboard UI
	•	Include:
	•	Left sidebar navigation
	•	Top header bar with search, notifications, profile
	•	Main content dashboard area
	•	Use a clean light theme with subtle gradients (white, soft gray, muted blue/green accents)
	•	Use modern typography (Inter / SF Pro) and consistent spacing
	•	Include cards, tables, charts, and AI insights panels

⸻

📌 Main Navigation (Sidebar)

Include 5 primary menu items with submenus:

⸻

1. Onboarding
	•	Employee onboarding pipeline
	•	Document collection (ID, contracts)
	•	Task checklist automation
	•	AI onboarding assistant (chat panel)

⸻

2. Hiring
	•	Candidate pipeline (Kanban board)
	•	Resume shortlisting (AI-powered scoring)
	•	Interview kit generator (questions, evaluation forms)
	•	Email automation + integrations (Gmail, Outlook)
	•	Calendar scheduling (Google Calendar integration)

⸻

3. HR Analytics
	•	Employee performance dashboards
	•	Attrition prediction (AI insights)
	•	Engagement metrics
	•	Workforce trends (charts & graphs)
	•	Natural language query input (Ask HR AI)

⸻

4. Team Formation
	•	Project requirements input panel
	•	AI-generated team recommendations
	•	Employee skill matrix visualization
	•	Bandwidth & availability tracking
	•	Drag-and-drop team builder UI

⸻

5. Compensation Management
	•	Salary and payroll dashboard
	•	Performance review integration
	•	Promotion & increment workflow
	•	AI-generated offer/promotion letters
	•	Automated email dispatch system
	•	Compensation benchmarking insights

⸻

🤖 AI-Native Features (Important)
	•	Include a visible AI Assistant panel (right side or floating)
	•	Show:
	•	“Ask HR Agent” input box
	•	Suggested prompts
	•	Generated insights cards
	•	Add automation indicators (e.g., “Auto-generated by AI” tags)

⸻

📊 Dashboard (Homepage)

Design a default overview dashboard showing:
	•	Key HR metrics (headcount, hiring pipeline, attrition rate)
	•	Recent AI insights (e.g., “High attrition risk in Engineering”)
	•	Tasks / alerts panel
	•	Quick actions (Hire, Promote, Assign Team)

⸻

🎨 Style Guidelines
	•	Clean enterprise UI (not flashy)
	•	Use soft shadows, rounded corners (8–12px)
	•	Use icons (Feather / Lucide style)
	•	Maintain high readability and spacing
	•	Focus on clarity + automation + intelligence

⸻

🚀 Optional Enhancements (if space allows)
	•	Activity timeline (AI actions performed)
	•	Notification center (automations triggered)
	•	Role-based access UI (Admin / Manager / Employee)
